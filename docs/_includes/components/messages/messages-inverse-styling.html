Inverse Styling

Add the class .message-inverse to any .message element to leverage inverse styling.

Message: This is a relevant message you should read. Information: This is a message that is more colorful than others. Success: You have done some action successfully and should be proud! Warning: Before going further, you may want to stop and read this. Danger: Something is wrong or potentially problematic, please read. 
<span class="message message-inverse">
  <strong>Information:</strong> This is a message that is more colorful than others.
</span>
