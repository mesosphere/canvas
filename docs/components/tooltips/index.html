Tooltips are an easy way to provide contextual tooltips or feedback related to a specific event, condition, or user action.

Use tooltips in your project to convey information, provide tips, communicate success, or even flag and notify the user of errors. Wrap your tooltip in an element with the class .tooltip to apply the default tooltip appearance. Any HTML can live inside a tooltip, though content is typically limited to text and simple actions.
