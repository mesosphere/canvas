CNVS includes a simple divider in both a normal and inverse style. alignment: left page-navigation: - label: Inverse Styling link: #dividers-inverse-styling - label: Spacing Modifiers link: #dividers-spacing-modifiers navigation: - label: Spacing Direction Modifiers link: #dividers-spacing-direction-modifiers ---

Use either the class .divider or go old-school with the <hr> tag to create a divider. Dividers fill the width of their parent containers. By default, dividers have space above and below. 

<!-- Alternatively, use <hr> -->
<div class="divider">
  …
</div>
